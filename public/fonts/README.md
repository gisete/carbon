# E-Ink Fonts

This directory contains custom fonts for the e-ink display screens.

## Required Font Files

Please add the following ChareInk font files to this directory:

- `ChareInk-Regular.ttf`
- `ChareInk-Bold.ttf`

These fonts are optimized for e-ink displays and will be used in the `/app/screens/` routes for improved legibility on the device.
